# go-docker
Building own docker from scratch to pull a container image from Docker Hub, unpack and run it in our own container runtime.

### Run Command
```
go run cmd/bin/main.go run <image-name> <commands>
```

### Build Command
```
go build cmd/bin/main.go
```
